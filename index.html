✓ SIN PERMISOS DE ADMIN
Solo necesitas Files.ReadWrite — cualquier usuario M365 puede otorgarlo sin intervención de TI.
📋 Instrucciones de configuración Azure AD (5 minutos)
  1. Ve a portal.azure.com → inicia sesión con tu cuenta Aramark
  2. Busca "App registrations" → "New registration"
  3. Nombre: PMO Central → Tipo: Accounts in this org only
  4. Tipo de app: Single-page application (SPA) → URI de redirección: la URL donde abres este archivo
  5. En Authentication → activa "Allow public client flows" = Yes
  6. En API Permissions → Add → Microsoft Graph → Files.ReadWrite (Delegated) — NO requiere admin
  7. Copia Client ID y Tenant ID → pégalos en los campos de arriba
  8. Haz clic en "Crear archivo Excel" → se crea TareasPMO.xlsx en tu OneDrive
